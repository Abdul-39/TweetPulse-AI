# TweetPulse AI 🚀

TweetPulse AI is an advanced NLP-based Tweet Sentiment & Hashtag Analyzer that uses BERT, RoBERTa, VADER, TextBlob, and LDA Topic Modeling to analyze Twitter data in real-time.

---

## 📌 Features

- Real-time Twitter sentiment analysis
- Hashtag trend analysis
- Topic modeling using LDA
- Multilingual tweet support
- BERT & RoBERTa integration
- CSV export support
- Interactive visualizations
- Word clouds and charts

---

## 🛠 Technologies Used

- Python
- Tweepy
- HuggingFace Transformers
- BERT / RoBERTa

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/TweetPulse-AI.git
```

### 2. Open Project

```bash
cd TweetPulse-AI/Source-Code
```

### 3. Install Requirements

```bash
pip install -r requirements.txt
```

### 4. Run Project

```bash
python app.py
```
